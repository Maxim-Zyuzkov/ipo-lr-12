# Лабораторная работа №12-13 по любимому питончику (шутка)

## Система транспортной логистики (Вариант 3)

### Запуск:
```bash
python3 main.py          # Консольная версия (ЛР12)
python3 main_gui.py      # Графическая версия (ЛР13)
```

### Функции:
1. Добавление клиентов и грузов
2. Добавление транспорта (фургоны и суда)
3. Оптимизация загрузки (VIP в первую очередь)
4. Минимизация используемого транспорта
